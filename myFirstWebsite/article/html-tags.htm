HTML tags

<a>
The anchor tag is used to indicate a link. The most useful attribute is href, which is used to tell the link where to go.
<base>
The base tag is used to define the main URL for ?.
<body>
The body tag is used to hold the content of a document. This content may be things like text and images.
<head>
The head tag is used to define the head of the document, and hold things like titles and meta information.
<div>
The div tag is used to separate sections of a document.
<h1>, <h2>, <h3>, ...
These heading tags are used to make a heading for content in the document.
<html>
The html tag is used to designate a document as an html document.
<img>
The image tag is used to put an image into the document. The most useful attribute is src, which is used to tell the document where the image comes from.
<li>
The line item tag is used to tell the document that the following text is an item in a list.
<link>
The link tag is used to link an external resource, such as a style sheet, to the document.
<meta>
The meta tag is used to give metadata about the document. Computers can use metadata, but the user cannot see it. Metadata is data about data.
<ol>
The ordered list tag is used to define a numbered list.
<p>
The paragraph tag is used to define a paragraph in the body of the document.
<script>
The script tag is used to define a script on the client's computer. An example script is JavaScript.
<ul>
The unordered list tag is used to define a bullet-point list.

Here is a sample HTML document which shows the use of each of these tags:

<!DOCTYPE html>       
<html>
	<head>
		<title></title>
	</head>
	<body>
	</body>
</html>
